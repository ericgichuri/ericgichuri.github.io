School System

About The System

This is a multivendor Web application system for a school. Each school can register to the system and have an account where they can do their school management. The system has features that are needed by a school. The system help keep records and easy management of school data.The system has a protected system user password and can also generate digital school IDs

Modules

  1. Student Module
  2. Staff Modules
  3. Examination Modules
  4. System Users
  5. Finance Modules
  6. Reporting Modules
  7. Books
  8. Catering & Accommondation
  9. Digital School ID generator software

Working of the System

This School system works on these steps:

  1. First you create/register school account
  2. Login to the school account and login as admin of the school
  3. You can register students, staffs, system users (users of the system)
  4. Curriculum can register Classes/ grades, Set Term, subject
  5. Examination module allows staff/ teachers to publish results also generate exam results forms
  6. In finance modules you can set term fee, update/add fee for students, generate fee statement, register project and project contribution, register expenditure
  7. Register books, update books
  8. Accommondation, catering modules for registering dormitory/ hostels and setting up catering timetable
  9. You can generate digital students IDs with QRcode embedded

Why You Need The System

There are various reason that makes this system unique:

  1. This system is unique becouse being a multivendor allows admin to customize the school data depending on what suites him well depeinding on levels
  2. You can track each fees, projects, and expenditure data.
  3. Digital School ID generated have QRcode which can be used in attendance systems
  4. The system is web base meaning you can work as long you have internet
  5. Easy management of school property and data.
