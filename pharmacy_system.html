Pharmacy System

About The System

This is a Web application system for a Pharmacy. The system has features that are needed by a pharmacy or a chemist. The system help keep records and easy management of pharmacy.The system has a protected system user password.The system has a chat system for employees

Modules

  1. Products and their categories
  2. Admin or Manager Modules
  3. Employee and System Users
  4. Cashier Modules
  5. Patients Modules
  6. Consultations
  7. Employee Chat Module

Working of the System

This pharmacy system works on these steps:

  1. This system allows add employees and system user and their roles like cashier, accountant, manager, system admin
  2. Allow to add products, categorically like pain killers, eye drops etc, and in terms of units like milliter(ml),miligrams(gm)
  3. Admin modules allows has a dashboard, finance and sales tracking and monitoring
  4. Cashier modules has adding product to cart and check out the cart, and a receipt is generated, also cashier can see his sales
  5. Consultation module allows to document certain disease symptoms into database for references
  6. Chat system allow employee to chat their co-workers
  7. Branch modules one can track branch sales for that day

Why You Need The System

There are various reason that makes this system unique:

  1. Branches Modules: The manager can track each branch sales to which branch yields more profit
  2. You can track each cahsier sales.
  3. Manager or Admin can deactivate System user account when on a leave(OFF)
  4. System Consultation, useful incase of unkwown patient symptoms, System user or Consultant can save the features in the Database for future references
  5. System Allow Employee to chat that can ease conversation
